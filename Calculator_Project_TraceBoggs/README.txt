CALCULATOR README:

-THIS CALCULTATOR PROGRAM INCLUDES THE BASIC 4 OPERATORS: +,-,/,*, AND PARENTHESIS.
-LIMITED TO ONLY WHOLE NUMBER INPUTS. (No negatives/decimals). IT CAN OUTPUT NON-WHOLE NUMBERS!
-IMPLEMENTS STACKS TO BALANCE PARENTHESIS AND TO CONVERT INFIX TO POSTFIX.
-----------------------------------------------------------------------------------------
-HARDWARE REQUIREMENTS:
N/A.

-SOFTWARE REQUIREMENTS:
ANY OPERATING SYSTEM AND C++/G++ TO COMPILE.(may be pre included depending on OS!)
-----------------------------------------------------------------------------------------
How to Compile:
1.Go to the terminal/command prompt for your given OS
2.change your directory to the calculator program's directory using the "cd" command and the directories name or PATH
3.compile by typing "c++ main.cpp calculator.cpp -o calculator"
-----------------------------------------------------------------------------------------
How to Run:
1.Once compiled, navigate to your calculator directory
2.IF on Windows, simply run the exe in your calculator directory
  IF on Linux(terminal), type ./calculator
-----------------------------------------------------------------------------------------
